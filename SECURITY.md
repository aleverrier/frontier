# Security Policy

This is research software for decoder experiments and reproducibility. It does
not provide a network service or process untrusted inputs as a security
boundary.

For security-sensitive reports, use a private GitHub security advisory if the
repository enables one, or contact the repository owner through GitHub.

No formal response SLA is declared.

Decoder correctness issues, numerical disagreements, benchmark regressions, and
reproducibility problems should be reported as ordinary issues rather than
security issues.
